Metrics and evaluation

The metrics take into account for the application are:

  • Accuracy: The accuracy of the model is the percentage of correct predictions. This is the most intuitive performance measure and it is simply a ratio of correctly predicted observation to the total observations. It is not the most reliable performance measure if the dataset is unbalanced.
  • F1 Score: The F1 score is the harmonic mean of precision and recall. It is a better measure than accuracy if the dataset is unbalanced.

The confusion matrix and the classification report are also used to evaluate the model and get more information about the performance of the model.

Accuracy

The accuracy of the model is 0.67. This means that the model is able to predict correctly the 67% of the cases.

This value is not very high, but we have to take into account that the type of task that the model is trying to solve is very difficult and it is possible that the dataset is not big enough to train a model that can predict correctly the 100% of the cases. To make sure that a Kickstarter campaign is going to be successful or not, there are many factors that have to be taken into account, and the dataset that we have used only contains a few of them.

F1 Score

The F1 score of the model is 0.65. This means that the model is able to predict correctly the 65% of the cases.

As we have said before, this value is not very high, but we have to take into account the same factors that we have talked about in the previous section.
